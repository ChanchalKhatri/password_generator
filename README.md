# 🔐 Password Generator

A modern and responsive Password Generator built using React and Vite. Generate strong and secure passwords with customizable options such as uppercase letters, lowercase letters, numbers, symbols, and password length.

## 🚀 Features

* Generate secure passwords instantly
* Adjustable password length
* Uppercase, lowercase, numbers, and symbols options
* Password strength indicator
* Copy password to clipboard
* Responsive design
* Smooth animations using Framer Motion

## 🛠️ Tech Stack

* React (Vite)
* JavaScript
* Tailwind CSS
* Lucide React
* Framer Motion

## 📦 Installation

```bash
git clone https://github.com/ChanchalKhatri/password_generator.git

cd password_generator

npm install

npm run dev
```
